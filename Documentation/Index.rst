=========
File List
=========

.. only:: html


	:Classification:
		file_list

	:Version:
		|release|

	:Language:
		en

	:Description:
		This extension provides a frontend plugin which shows a list of files and folders in a specified directory
		on the file system (comparable to Apache's directory listing).
